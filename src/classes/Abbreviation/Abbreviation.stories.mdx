# Abbreviation

The Abbreviation class is used with `<abbr>` elements. Use the `<abbr>` tag around abbreviations, which should be defined in the surrounding text. The `<abbr>` element should also have a `title` attribute giving the expanded version of the abbreviation.

<Canvas>
  <p>The acronym <abbr class='abbr' title='Digital Linguistics'>DLx</abbr> stands for 'Digital Linguistics'.</p>
